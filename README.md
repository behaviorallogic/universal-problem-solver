The Universal Problem Solving Algorithm
=======================================

This repository is a companion to my Substack article.

I believe there is one — and only one — way to solve a problem. The process is imperfect, with plenty of places to fail, but I've found nothing that works any better. At its core, it goes like this:

1. Choose the criteria for a good solution
2. Generate possible solutions
3. Evaluate them against the criteria
4. Choose the best one. If none are good enough, return to step 2.

The [sqrt.html](sqrt.html) file demonstrates a few examples of finding the square root of 2 using this basic approach.
